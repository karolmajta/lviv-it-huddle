Install a text editor suitable for programming if you don't have one yet. You can grab one on atom.io.

Download a template project and extract it somewhere on your hard drive.

Open the editor, and open the whole project (CTRL+SHIFT+O if you're using atom, then just point the directory where you extracted the template). You should see two files:

  • index.html
  • game.js
In your editor, you can ignore index.html and just focus on game.js. This is where you'll program your grame.

Open your web browser (preferrably Chrome), and open the index.html file (CTRL+O and then point at the file).

Make sure you see the welcome message in your browser. In your editor try changing the welcome text from "Hello Lviv" to something you like ("Hello Kittens"?), save the changes. Refresh contents of your browser. Make sure you see things changed.
